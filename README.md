# 処理

```
$ cd
$ git clone https://github.com/DialBird/dotfiles.git
$ cd dotfiles && ./setup.sh

# pureというcolorテーマを使っている
$ npm install --global pure-prompt

# 必要なagとfzfをインストール
$ brew install ag fzf
$ $(brew --prefix)/opt/fzf/install
```
